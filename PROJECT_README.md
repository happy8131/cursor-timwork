# CINK 웹 도면뷰어

건설 현장 도면 탐색 인터페이스 프로토타입

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 접속

### 도면 이미지 설정

`public/drawings/` 폴더에 도면 이미지 파일을 넣어주세요.  
`metadata.json`에 정의된 파일명과 일치해야 합니다.

## 기술 스택

- **프레임워크**: Next.js 16 (App Router)
- **언어**: TypeScript
- **UI**: React 19 + Tailwind CSS 4 + shadcn/ui
- **상태 관리**: Zustand
- **도면 뷰어**: react-zoom-pan-pinch (줌/팬/핀치)
- **아이콘**: Lucide React

## 구현 기능

- [x] **도면 탐색** - 좌측 사이드바에서 건물/공종/리비전 계층 탐색
- [x] **도면 표시** - 선택한 도면 이미지를 확대/축소/패닝으로 확인
- [x] **컨텍스트 인식** - 상단 브레드크럼 + 사이드바 하이라이트로 현재 위치 표시
- [x] **검색** - 도면명, 공종, 리비전 설명 등 키워드 검색
- [x] **리비전 타임라인** - 시간순 리비전 이력 및 변경사항 확인
- [x] **도면 상세 정보** - 하단 패널에서 선택 도면의 메타데이터 확인
- [x] **전체 배치도 인터랙션** - 배치도에서 건물 영역 클릭 시 해당 동으로 이동
- [x] **반응형 디자인** - 모바일/태블릿/데스크톱 대응
- [x] **키보드 단축키** - `/` 검색, `Ctrl+B` 사이드바 토글
